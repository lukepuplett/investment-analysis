# Microsoft Corporation (MSFT)

## Company Overview
Microsoft Corporation is a technology company that develops and supports software, services, devices, and solutions worldwide. The company operates through three segments: Productivity and Business Processes (Office, LinkedIn, Dynamics), Intelligent Cloud (Azure, server products, enterprise services), and More Personal Computing (Windows, devices, gaming, search advertising).

**Ticker:** MSFT  
**Exchange:** NASDAQ  
**Sector:** Technology  
**Industry:** Software—Infrastructure / Cloud / Productivity  

## Executive Summary (BLUF - Bottom Line Up Front)

**Consolidated BLUF (2025-02-23):** Microsoft delivered Q2 FY25 revenue of **$69.6B (+12% YoY)** and TTM revenue of **$281.7B** with industry-leading profitability (operating margin **46.7%**, net margin **38.1%**, FCF **$84.5B** TTM). Microsoft Cloud exceeded **$40B** quarterly (+21% YoY); AI revenue run rate exceeded **$13B** (+175% YoY). Strong balance sheet (~$90B cash and ST investments) and **$9.7B** capital return in the quarter. **Hold/Buy** for long-term investors; Base Case fair value ~$430/share; key risks are cloud competition and regulatory scrutiny.
