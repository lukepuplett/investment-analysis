# Regime C: Platform Computing Model

**Core Thesis:** Robotaxis replace 50%+ car ownership. Foundational coordination layer. Winner-take-most.

## Economics Snapshot

| Parameter | Value | Rationale |
|-----------|-------|-----------|
| **TAM** | 40–50% of personal miles + new uses | Replaces car ownership + logistics explosion |
| **Fleet size** | 10–50M vehicles nationally | 50% of 290M car fleet + redundancy |
| **Ownership substitution** | 50%+ | Majority car ownership collapse |
| **Take-rate** | 40–65% | Ownership replacement: cheaper than owning, premium pricing justified |
| **Trip intensity** | 150–300+ trips/capita/yr | Kids, elderly, micromobility, same-day delivery |
| **Induced demand** | 10–25x | Mobility becomes primary consumption good |
| **Regulatory approval** | Exponential post-inflection | Human driving seen as dangerous |
| **Competitive outcome** | Winner-take-most | Software scaling, network effects, generalization |
| **Terminal multiple** | 50–150x | Platform monopoly + multiple revenue streams |
| **2035 Waymo revenue** | $50–150B platform | $2–10T terminal value |
| **2035 Tesla revenue** | Loser or niche | $50–200B (if survives) or $0 (if not) |

## What This Assumes (all must hold together)

✓ Autonomy truly robust (no edge case plateau)  
✓ Ownership drops 50%+ in major metros  
✓ Regulatory S-curve inflects (then exponential)  
✓ Stack generalizes across regions/companies  
✓ Marginal costs decline 30%+ annually (software dynamics)  
✓ Winner achieves 70%+ market share  
✓ Adjacent revenue (non-ride) exceeds 30%  
✓ Network effects compound (more vehicles → better AI → faster improvement)  

## If Any Break:

**Fall to B:**
- Ownership tops out <30% → replacement not systemic
- Approval stays linear → bureaucratic, not exponential
- Second player scales competitively → software isn't scaling
- Take-rates compress <30% → margins don't support ownership replacement
- Intervention rate plateaus → autonomy fragile

**Fall to A:**
- Intervention rate stalls completely
- Ownership substitution fails to emerge
- Regulatory approval reversed (major incident)
- Competition intensifies (margins collapse)

## Internal Coherence Check

Fleet 10–50M + 40–65% take-rate + 150–300 trips/capita = consistent only if:
- Ownership replacement 50%+ happening
- Software scaling evident (costs declining, winner emerging)
- Regulatory environment fundamentally changed (exponential approval)
- Stack generalizes (knowledge transfer across regions)

**If fleet stays <2M:** ownership not really replaced → incoherent → fall to B

**If take-rates stuck 25–35%:** margins don't support customer switching from ownership → incoherent → fall to B

**If approval stays linear:** regulatory environment unchanged → incoherent → fall to B

**If both players scale equally:** software isn't scaling, not winner-take-most → incoherent → fall to B
