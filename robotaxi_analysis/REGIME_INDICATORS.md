## Regime C: Stays If All Hold

**Baseline Metrics (Confirm Regime C)**

| Metric | Range | If Broken → Shift To |
|--------|-------|---------------------|
| Ownership substitution | 40–60%+ in metros | <30% → B |
| Approval pace | >10%/year exponential | <8%/year sustained → B |
| Non-ride revenue | >30% of miles | <20% → B |
| Take-rate | 40–65% resilient | <35% → B |
| Fleet deployment | 5M–50M vehicles | <2M → B |
| Software scaling | 30%+/year cost decline | <15%/year → B |
| Winner consolidation | 70%+ market share | <50% (second player matching) → B |
| Trip intensity | 150–300+/capita | <100/capita → B |

**Fragility Test 1: Autonomy Stalls**
- If intervention rate flat 4+ quarters (autonomy not improving), stack doesn't generalize
- Cost decline stalls below 20%/year
- Second player can match leader
- Regime C → Regime B downshift

**Fragility Test 2: Ownership Plateaus**
- If ownership growth stops <40% (even with fleet >5M), network effects not compounding
- Regulatory narrative stalls (not exponential)
- Regime C → Regime B downshift

**Catastrophe Test: Major Incident**
- Fatality, multi-vehicle crash, regulatory backlash
- Ownership reverses (e.g., 45% → 30% → 15%)
- Approval reversed (pilot closure)
- Regime C → Regime A collapse

## Leading Indicator Hierarchy (Monitor in Order)

### Tier 1: Ownership Substitution (Most Important)
**Why:** Unlocks all other variables. If true, take-rates stay high, approval accelerates, fleet scales. If false, margins compress, approval bureaucratic.

**Signal A:** <5% in all metros by Q4 2027 → Confirm Regime A
**Signal B:** 15–30% in major metros by Q4 2027 → Confirm Regime B
**Signal C:** 35–50% in 2+ metros by Q4 2028 → Regime C likely

**Data sources:** Census, insurance dropoff, public transit ridership, Waymo/Tesla ridership surveys

---

### Tier 2: Regulatory Approval Pace (Second Most Important)
**Why:** Must align with ownership and autonomy progress. If approval faster than ownership changes, narrative mismatch. If slower, infrastructure case fails.

**Signal A:** Approval 2–6%/year sustained → Confirm Regime A (bureaucratic)
**Signal B:** Approval accelerating to 5–8%/year by 2027 → Confirm Regime B (S-curve inflection)
**Signal C:** Approval >10%/year by 2028 → Regime C signals (exponential)

**Data sources:** Regulatory filings, new metro approvals per quarter, agency commentary

---

### Tier 3: Non-Ride Revenue Mix (Third Priority)
**Why:** Reveals if asset layer (logistics, delivery, ads) is monetizing. Regime C requires 30%+ of miles non-ride to justify capital.

**Signal A:** <5% of miles by Q4 2027 → Confirm Regime A (pure rideshare)
**Signal B:** 10–25% of miles emerging by Q4 2027 → Confirm Regime B (infrastructure layer forming)
**Signal C:** >30% of miles by Q4 2028 → Regime C (platform layer exceeds core ride)

**Data sources:** Operator press releases, investor presentations, fleet telemetry analysis

---

### Tier 4: Take-Rate Trajectory (Fourth Priority)
**Why:** Derivative of ownership + competition. If ownership replacing cars, take-rates stay 35%+. If not, Uber re-enters, rates compress to 18%.

**Signal A:** 18–32% stable with Uber/Lyft pressure → Confirm Regime A
**Signal B:** 25–40% sustainable without demand erosion → Confirm Regime B (ownership premium evident)
**Signal C:** 40–65% resilient despite fleet scaling to 5M+ → Regime C (lock-in + network effects)

**Data sources:** Earnings reports, pricing analysis, gross revenue / miles

---

### Tier 5: Fleet Guidance (Supports Other Signals)
**Why:** Company guidance on fleet size reveals internal TAM model.

**Signal A:** <300k fleet target → Regime A (rideshare niche)
**Signal B:** 500k–2M fleet target → Regime B (owns ownership replacement assumption)
**Signal C:** >5M fleet target → Regime C (owns software scaling + majority ownership)

**Data sources:** Earnings guidance, investor presentations, capex plans

## Why These Metrics Matter More Than Others

**Not tracking:**
- Revenue growth % (derivative of fleet × utilization × take-rate)
- Utilization % (follows ownership substitution and approval)
- Competitive gap (follows software scaling rate, which follows regime)
- Gross margin % (follows take-rate, which follows ownership)

**These follow. The leading indicators are:**
1. Ownership substitution % (TAM signal)
2. Approval pace (regulatory narrative + autonomy proof)
3. Non-ride revenue mix (platform layer formation)
4. Take-rate resilience (ownership replacement evidence)
5. Fleet guidance (internal TAM model)

**Only need 3–4 of these in Tier 1–2 to force regime shift. Don't wait for all to align—if any hard trigger hits, regime is dead.**
