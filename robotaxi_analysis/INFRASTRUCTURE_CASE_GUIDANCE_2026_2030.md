# Infrastructure Case Baseline Guidance: 2026–2030

## Overview

This document provides quarterly baseline guidance for the **Infrastructure Scenario**, anchoring investor expectations for comparing against actual quarterly results. The Infrastructure case assumes:
- Behavior change inflection **starts 2028** (land use shifts, second-car elimination, new user cohorts)
- City integration monetization **begins 2027** (parking, emergency, logistics value layers)
- Terminal 2035 outcome: Waymo **165k vehicles, $6.1B platform revenue, 28-32x multiple = $170B terminal value**

**Use:** Track actual quarterly results against this guidance. Deviations >1.5 sigma warrant scenario repricing.
