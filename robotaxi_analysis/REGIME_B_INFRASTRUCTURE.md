# Regime B: Infrastructure Layer Model

**Core Thesis:** Robotaxis capture 20–30% car ownership. Partial TAM shift. Essential mobility layer.

## Economics Snapshot

| Parameter | Value | Rationale |
|-----------|-------|-----------|
| **TAM** | 15–20% of personal miles | Rideshare + transit + partial ownership |
| **Fleet size** | 500k–2M vehicles | Dense coverage for ownership replacement |
| **Ownership substitution** | 20–30% | Meaningful shift, not majority |
| **Take-rate** | 25–40% | Higher margins (reduced competition, convenience premium) |
| **Trip intensity** | 50–120 trips/capita/yr | Kids, elderly, micromobility enabled |
| **Induced demand** | 3–5x | Behavior change from access |
| **Regulatory approval** | S-curve 2028–2030 | Inflection when safety undeniable |
| **Competitive outcome** | Winner-leaning or bifurcated | Not yet winner-take-all |
| **Terminal multiple** | 25–50x | Infrastructure-like (essential) + moat |
| **2035 Waymo revenue** | $15–30B platform | $150–400B terminal value |
| **2035 Tesla revenue** | $8–15B platform | $80–200B terminal value |

## What This Assumes (all must hold together)

✓ Ownership drops 20–30% in major metros  
✓ Regulatory inflection happens (S-curve, not linear)  
✓ Kids/elderly become primary users  
✓ Logistics/delivery becomes 10–25% of miles  
✓ Take-rates expand (convenience premium)  
✓ Adjacent monetization emerges (subscriptions, data)  
✓ Marginal costs decline moderately (20–30%)  

## If Any Break:

**Fall back to A:**
- Ownership stays <5% despite availability → not really substituting
- Approval stays <3%/year → bureaucratic, not civilization-scale
- Intervention rate plateaus → autonomy fragile
- Take-rates compress <15% → competition persists

**Escalate to C:**
- Ownership >30% in major metro → replacement deeper than B
- Approval >10%/year sustained → S-curve inflection, exponential
- Non-ride revenue >30% of total → platform dynamics emerging
- Fleet guidance >2M → assuming ownership replacement massive

## Internal Coherence Check

Fleet 500k–2M + 25–40% take-rate + 50–120 trips/capita = consistent only if:
- Ownership replacement 20–30% happening
- Regulatory inflection visible (approval accelerating)
- Non-ride use cases emerging (10–25% miles)

**If fleet stays <300k:** ownership not being replaced → incoherent → fall to A

**If take-rates compress to 15%:** competition hasn't reduced, margins unsustainable → fall to A

**If approval stays linear:** regulatory environment unchanged, not civilization-scale → fall to A

**If fleet jumps >2M:** ownership replacement deeper, software scaling evident → shift to C
