# Regime A: Utility Transport Model

**Core Thesis:** Robotaxis compete with Uber/Lyft. Remain niche vs. car ownership.

## Economics Snapshot

| Parameter | Value | Rationale |
|-----------|-------|-----------|
| **TAM** | 5% of personal miles | Rideshare + taxi only |
| **Fleet size** | 120–300k vehicles | Serves rideshare demand |
| **Ownership substitution** | <5% | People keep cars |
| **Take-rate** | 18–32% | Competitive pressure from Uber |
| **Trip intensity** | 12–45 trips/capita/yr | Anchored to rideshare behavior |
| **Induced demand** | 1.35–2.25x | Modest elasticity |
| **Regulatory approval** | 3–6%/year linear | Local transport authority |
| **Competitive outcome** | Waymo + Tesla coexist | Stable market split |
| **Terminal multiple** | 9–18x | Transport commodity + moat |
| **2035 Waymo revenue** | $3–5B platform | $30–60B terminal value |
| **2035 Tesla revenue** | $4.2B platform | $15–40B terminal value |

## What This Assumes (all must hold together)

✓ Ownership stays near-universal  
✓ Approval stays bureaucratic/linear  
✓ Kids/elderly stay car-dependent  
✓ Logistics remains separate from robotaxi  
✓ Competition prevents margin expansion  
✓ No software-like scaling effects  
✓ No adjacent monetization (ads, data, subscriptions)  

## If Any Break:

- Ownership >10% in deployment metro → **Shift to Regime B**
- Approval 2x baseline sustained → **Shift to Regime B**
- Non-ride use cases >10% miles → **Shift to Regime B**
- Take-rates >25% without pressure → **Shift to Regime B**

## Internal Coherence Check

Fleet 120–300k + 18–32% take-rate + 12–45 trips/capita = consistent only if:
- Ownership substitution minimal (<5%)
- Competition prevents margin expansion
- Approval stays gated (regulatory, not market demand)

**If fleet jumps to 1M:** ownership must be replacing cars → incoherent with Regime A → shift to B/C

**If take-rate hits 40%:** margins support ownership replacement → incoherent with <5% substitution → shift to B/C

## Monitoring (Stay in Regime A if):

| Metric | Target | Frequency |
|--------|--------|-----------|
| Ownership substitution | <5% | Quarterly |
| Approval pace | 3–6%/year | Quarterly |
| Non-ride mix | <5% of miles | Quarterly |
| Take-rate | 18–32% | Quarterly |
| Fleet targets | <300k | Annual |
| Intervention rate | Improving steadily | Quarterly |

**Fallback to A if:** Autonomy stalls (intervention rate flat 4+ quarters)
